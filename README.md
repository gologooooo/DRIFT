# Model Introduction
DRIFT is a novel topology-decoupled framework designed for robust out-of-domain spatiotemporal and multivariate time series forecasting.

# Dependencies
This repository provides only the model implementation and training scripts for DRIFT. The entire project is built upon the BasicTS framework. BasicTS provides the essential underlying infrastructure. To run the code in this repository, you must first set up the BasicTS environment.

Installation Steps:

1.Set up BasicTS: Please clone the official BasicTS repository and follow its installation instructions: https://github.com/GestaltCogTeam/BasicTS

2.Integrate DRIFT Code: Clone this DRIFT repository and integrate the provided model files and configuration files into your BasicTS project structure.
